git 删除本地分支和远程分支。本地代码回滚和远程代码库回滚

2015-05-04

【git 删除本地分支】
git branch -D br

【git 删除远程分支】

git push origin :br (origin 后面有空格)

git代码库回滚: 指的是将代码库某分支退回到以前的某个commit id
【本地代码库回滚】：
git reset –hard commit-id :回滚到commit-id，讲commit-id之后提交的commit都去除
git reset –hard HEAD~3：将最近3次的提交回滚
